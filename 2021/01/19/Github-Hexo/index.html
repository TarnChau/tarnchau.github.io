Blog configuration: Github + Hexo

Posted on 2021-01-19 Edited on 2021-01-21 In Config Views:

Use github and hexo to config your own website.

Install Git and Connect Github

windows

(1) Download “Git” from Official website: https://git-scm.com
(2) Install “*.exe” file, choose “add to path”
(3) Check if installed successfully: in “cmd”

(4) right-click menu: “Git Bash Here”
(5) Set “user.name” and “user.email”: (replace with your own infomation)

(6) Generate “ssh-keygen”: (press “Enter”)

(7) Find “id_rsa.pub” in folder: “C:\Users"user_name”.ssh”, open and copy all
(8) Paste the codes in: “New SSH key”, https://github.com/settings/ssh/new, “Title” as you wish
(9) Check the connection in “Git Bash”

Install Node.js

(2) Install “*.exe” file (folder for example: “E:\Hexo\nodejs”), choose “add to path”
(3) Check if installed successfully: in “cmd”

(4) New folders in “E:\Hexo\nodejs” (example): “node_cache” and “node_global”

(6) Close “cmd” and config the “Environment Variables”: (1) “system variable-path-E:\Hexo\nodejs”, (2) “user variable-path-E:\Hexo\nodejs\node_global” (manual prefix path)
